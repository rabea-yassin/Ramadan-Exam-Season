# Changelog

All notable changes to Command Center will be documented in this file.

## [1.0.0] - 2026-03-12

### Added
- Initial release
- Dashboard with exam countdown and progress tracking
- Backlog for master task list
- Daily checklist with task pinning
- Pomodoro timer with break timer
- Timer persistence across page navigation
- 7-day activity heatmap
- Watch Later video queue
- Battle Log daily journal
- File-based data storage (data.json)
- Weighted progress by topic importance
- Keyboard shortcuts (Space bar for Pomodoro)

### Features
- 3-layer task system (Backlog → Checklist → Dashboard)
- Real-time progress calculation
- Study minutes tracking
- Pomodoro session counter
- Break time tracking
- Streak system
- Responsive design
- Dark sidebar navigation

### Technical
- Node.js + Express backend
- Vanilla JavaScript frontend
- RESTful API
- JSON file storage
- No build step required
